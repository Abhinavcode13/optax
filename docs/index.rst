Optax
-----

Optax is a gradient processing and optimization library for JAX.
It has been designed to facilitate research by providing building blocks that
can be easily recombined to implement new optimizers based on existing ideas.

Installation
------------

To install Optax, you can use ``$ pip install optax``.
Optax is built on JAX. See `here <https://github.com/google/jax#pip-installation-cpu>`_
for instructions on installing JAX.
